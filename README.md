## 反馈问题

- 如果有红包先花掉再开脚本，不然可能需要支付密码
- 出现下单地址不是默认地址的，在线下一单，取getOrderInfo.action链接的cookie
- CMD界面卡住、关闭CMD的快速编辑模式就行了
